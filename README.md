## Building

> With yarn

```bash
git clone https://github.com/Kayque-Goncalves/saturn-music-web.git
$ cd saturn-player
$ yarn install
$ yarn start
```

> With npm

```bash
git clone https://github.com/Kayque-Goncalves/saturn-music-web.git
$ cd saturn-player
$ npm install
$ npm run start
```

Runs the app in the development mode.<br/>
